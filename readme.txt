== Description ==  
Jobs Manager is a simple plugin that allows you to create and manage job listings within WordPress.  
- Adds a "Jobs" custom post type (CPT).  
- Includes custom fields for job details (Company Name, Location, Salary, Description).  
- Uses a `[job_listings]` shortcode to display job listings anywhere.  
- Provides an admin settings page to set a default salary.  

== Installation ==  
1. Download the plugin ZIP file and upload it to `/wp-content/plugins/`.  
2. Activate the plugin through the "Plugins" menu in WordPress.  
3. Go to "Jobs" in the WordPress admin panel to add new job listings.  
4. Use `[job_listings]` on any page to display the jobs.  

== Frequently Asked Questions ==  
= How do I change the default salary? =  
- Navigate to **Settings > Jobs Settings** and enter a default salary value.  

= Can I customize the job listing display? =  
- You can override the plugin's CSS or modify the shortcode output in `class-jm-shortcodes.php`.
